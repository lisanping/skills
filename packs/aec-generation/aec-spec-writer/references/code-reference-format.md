# 规范引用格式约定

> 何时读取:在任何设计说明 / 规格书中引用规范、标准、图集时。

## 国标 / 行标 / 地标 引用格式

**完整格式**:`《名称》编号-年份 (修订版本)`

正确示例:
- `《建筑设计防火规范》GB 50016-2014 (2018 年版)`
- `《民用建筑设计统一标准》GB 50352-2019`
- `《住宅设计规范》GB 50096-2011`
- `《建筑抗震设计规范》GB 50011-2010 (2016 年版)`
- `《混凝土结构设计规范》GB 50010-2010 (2015 年版)`
- `《无障碍设计规范》GB 50763-2012`
- `《建筑内部装修设计防火规范》GB 50222-2017`

错误示例:
- ❌ `按防火规范要求` —— 无编号
- ❌ `GB 50016` —— 无年份
- ❌ `GB 50016-2006` —— 已废止

## 国际标准引用格式

- **ASTM**:`ASTM C 1396 / C 1396M — Standard Specification for Gypsum Board`
- **ISO**:`ISO 6892-1:2019 — Metallic materials — Tensile testing`
- **ACI**:`ACI 318-19 — Building Code Requirements for Structural Concrete`
- **AISC**:`AISC 360-22 — Specification for Structural Steel Buildings`

## 不确定时的处理

如规范现行版本无法确认:
1. 标注 `[待核实版本]`,例如 `《XXX 规范》GB XXXXX-[待核实版本]`
2. 在文档末尾"待确认事项"列出
3. **绝不**编造一个"看起来合理"的年份
