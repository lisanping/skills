## 编号约定

| 前缀 | 部位                    |
| ---- | ----------------------- |
| `L`  | 楼地面(Floor)           |
| `W`  | 内墙面(Wall - interior) |
| `EW` | 外墙面(Exterior Wall)   |
| `C`  | 顶棚(Ceiling)           |
| `R`  | 屋面(Roof)              |
| `B`  | 踢脚(Base)              |

## 做法层次书写规则

1. **顺序固定**:
   - 楼地面 / 屋面 → **自上而下**(面层 → 找平 → 防水 → 保温 → 结构层)
   - 墙面 / 顶棚 → **自外向内**(饰面 → 基层 → 龙骨 / 抹灰 → 结构)
2. 每一层注明 **厚度 + 材料 + 强度 / 配比**(若适用)
3. 防水层、保温层必须注明产品类型(SBS / APP、XPS / EPS / 岩棉)及厚度
4. 燃烧性能等级标注:A / B1 / B2 / B3,引用《建筑材料及制品燃烧性能分级》GB 8624-2012

## 与房间表的对应

做法表必须与房间表(房间号 → 装修编号)成对出现,避免"做法定义了但没用"或"房间用了但没定义"的情况。
