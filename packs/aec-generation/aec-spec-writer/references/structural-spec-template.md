# 结构施工图设计说明 — 模板骨架

> 何时读取:生成结构施工图设计说明时。

## 顶层章节(顺序不可调整)

```markdown
# [项目名称] 结构施工图设计说明

## 一、工程概况
- 建筑类别 / 结构形式(框架 / 剪力墙 / 框剪 / 钢结构等)
- 建筑层数 / 建筑高度 / 总建筑面积
- 设计使用年限 / 结构安全等级 / 地基基础设计等级

## 二、设计依据
1. 国家及地方现行规范:
   - 《建筑结构荷载规范》GB 50009-2012
   - 《混凝土结构设计规范》GB 50010-2010 (2015 年版)
   - 《建筑抗震设计规范》GB 50011-2010 (2016 年版)
   - 《高层建筑混凝土结构技术规程》JGJ 3-2010
   - 《建筑地基基础设计规范》GB 50007-2011
   - 《钢结构设计标准》GB 50017-2017
2. 岩土工程勘察报告(报告编号 / 勘察单位 / 日期)
3. 建筑专业提供的图纸版本

## 三、自然条件
- 抗震设防烈度 / 设计基本地震加速度 / 设计地震分组 / 场地类别
- 基本风压 / 地面粗糙度类别
- 基本雪压
- 场地冻土深度 / 地下水位

## 四、荷载取值
- 楼面均布活荷载(按使用功能列表)
- 屋面活荷载 / 雪荷载
- 风荷载体型系数与放大系数
- 特殊荷载(设备、堆载、吊装)

## 五、主要结构材料
- 混凝土强度等级(分构件类别)
- 钢筋种类与级别
- 钢材牌号(钢结构项目)
- 砌体材料 / 砂浆等级

## 六、地基与基础
- 地基持力层 / 承载力特征值 fak
- 基础类型(独立 / 条形 / 筏板 / 桩基)
- 桩型 / 桩径 / 桩长 / 单桩承载力

## 七、上部结构
- 抗震等级(按构件类别分别注明)
- 楼盖体系
- 关键构造措施(剪力墙边缘构件、框架节点核心区箍筋加密)

## 八、构造要求
- 钢筋锚固与搭接长度表
- 保护层厚度
- 后浇带 / 沉降缝 / 抗震缝设置

## 九、施工注意事项
- 模板与支撑
- 混凝土浇筑顺序与养护
- 预留预埋
- 与其他专业接口

## 十、其他
- 图集索引
- 未尽事宜按现行规范执行
```

## 填写规则

1. 抗震等级、混凝土强度、钢筋级别等关键参数**必须由结构专业工程师确认**,SKILL 不得自行设定
2. 引用图集格式:`图集号-页码`,例如 `16G101-1 P67`
3. 数值缺失时用 `[由结构专业填写]` 占位,不要猜
