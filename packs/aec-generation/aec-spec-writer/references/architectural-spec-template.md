# 建筑施工图设计说明 — 模板骨架

> 何时读取:生成国标体系下的建筑施工图设计说明时。

## 顶层章节(顺序不可调整)

```markdown
# [项目名称] 建筑施工图设计说明

## 一、工程概况
- 项目名称 / 建设地点 / 建设单位
- 用地面积 / 总建筑面积 / 地上 / 地下建筑面积
- 建筑层数(地上 / 地下) / 建筑高度
- 结构形式 / 设计使用年限 / 抗震设防烈度
- 耐火等级 / 屋面防水等级 / 地下室防水等级
- 主要功能分区

## 二、设计依据
1. 建设单位提供的设计任务书及相关批复文件
2. 国家及地方现行规范、标准:
   - 《建筑设计防火规范》GB 50016-2014 (2018 年版)
   - 《民用建筑设计统一标准》GB 50352-2019
   - 《建筑结构荷载规范》GB 50009-2012
   - 《建筑地面设计规范》GB 50037-2013
   - 《建筑内部装修设计防火规范》GB 50222-2017
   - 《无障碍设计规范》GB 50763-2012
   - [按项目类型补充地方标准]
3. 上一阶段(方案 / 初步设计)审查意见

## 三、设计标高与定位
- ±0.000 相对绝对标高
- 各层标高表
- 建筑定位坐标依据

## 四、墙体工程
- 外墙:材料 / 厚度 / 保温构造 / 饰面
- 内墙:分户墙 / 隔墙材料与厚度
- 砌筑砂浆等级、构造柱与圈梁要求

## 五、楼地面 / 屋面 / 门窗 / 装修工程
- 引用"室内装修做法表"(见第八章)
- 屋面构造层次(自上而下)
- 门窗表索引及性能要求(气密、水密、抗风压、传热系数)

## 六、防水、保温、防火构造
- 卫生间 / 厨房 / 屋面防水做法及附加层
- 外墙 / 屋面 / 地面保温做法及厚度
- 防火分区 / 防火门窗 / 防火封堵

## 七、无障碍设计
- 无障碍出入口 / 通道 / 卫生间 / 电梯
- 引用《无障碍设计规范》GB 50763-2012 条款

## 八、室内外装修做法表
| 部位 | 做法编号 | 构造层次(自外向内) | 备注 |
| ---- | -------- | ------------------ | ---- |

## 九、节能设计专项说明
- 体形系数 / 窗墙比
- 围护结构传热系数限值与设计值对照
- 引用节能标准条款

## 十、施工注意事项
- 一般要求
- 特殊部位施工要求
- 与其他专业的接口
```

## 填写规则

1. **章节序号不可调整**,即使某项不适用,保留标题写"本工程不涉及"
2. **规范引用**严格按 `code-reference-format.md` 格式
3. 所有数值带单位;面积 m²、长度 mm(室内)/ m(总图)、传热系数 W/(m²·K)
4. **签字栏**(设计 / 校对 / 审核 / 审定 / 项目负责人)在文末预留,不得代签
