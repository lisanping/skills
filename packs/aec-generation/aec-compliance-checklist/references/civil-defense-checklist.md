## 二、CD-1　防护等级与防护单元

### 检查目的

防护等级和单元划分是后续所有条款的前提。划错单元会导致后续口部、通风、防护构件全部失效。

### 检查项

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| CD-1.1 | GB 50038 3.1 | 防护等级与审批意见书一致 | 与意见书参数完全一致 | | | 是否甲类、乙类需明确 |
| CD-1.2 | GB 50038 3.2.1 | 防护单元最大允许面积 | 各等级和功能的限值不同（如 5 级人员掩蔽 ≤ 2,000m²） | | | 一旦超出须分单元 |
| CD-1.3 | GB 50038 3.2.2 | 抗暴单元划分 | 防护单元内每个抗暴单元的面积按规范要求 | | | 抗暴单元在防护单元内进一步细分 |
| CD-1.4 | GB 50038 3.2.3 | 单元间防护密闭隔墙 | 由钢筋混凝土构成，厚度按抗力级别 | | | 隔墙厚度是常错点 |
| CD-1.5 | GB 50038 3.2 | 上下层防护单元对位 | 多层人防地下室的防护单元应上下对位 | | | 错位会导致防护墙力学传力问题 |

## 四、CD-3　口部设计

### 检查目的

口部是防护工程"破坏概率最高"的位置。一个口部失效，整个防护单元失效。

### 口部主要构件

| 构件 | 作用 | 设置位置 |
|------|------|---------|
| 防护密闭门（FHM） | 抵御冲击波 + 密闭，承受爆炸荷载 | 主要出入口、连通口、风口外侧 |
| 密闭门（MM） | 仅密闭，不抗爆 | 主要出入口防护密闭门后的第二道 |
| 防爆波活门（FHHM） | 进/排风口的抗爆密闭装置 | 通风进/排风井口 |
| 悬摆活门 | 用于油机排烟 | 油机排烟井 |

### CD-3.1 主要出入口

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| CD-3.1.1 | GB 50038 3.4 | 每个防护单元主要出入口数量 | 至少 1 个直通室外的主要出入口 | | | 主出入口必须独立直通室外 |
| CD-3.1.2 | GB 50038 3.4 | 主要出入口形式 | 室外开敞楼梯、坡道、竖井（按规范选用） | | | 室外楼梯最常见 |
| CD-3.1.3 | GB 50038 3.4 | 主要出入口疏散宽度 | 按掩蔽人数计算且不小于规范最小值 | | | |
| CD-3.1.4 | GB 50038 3.5 | 防护门 + 密闭门的组合 | 主出入口防护密闭门 1 道 + 密闭门 1 道（含洗消间或防化通道时增加） | | | 防化乙级以上一般要求洗消间 |
| CD-3.1.5 | GB 50038 3.5 | 门洞净宽与净高 | 按通行人数和家具设备进出需求；最小净宽 0.80m | | | 战时人流大时需放宽 |

## 十一、常见错误（与人防专项相关）

1. **未查阅修建审批意见书就开始审查**——审批意见书是项目实际防护等级的法定依据，没有它的审查只是纸面练习。
2. **防护单元面积超限**——经常发生在地下车库人防区，设计师只想着停车位数量，忘了人防面积上限。
3. **口部位置不当**——两个口部布置在同一方向，违反"分别通向不同方向"。
4. **防护密闭门开启方向错误**——往非密闭区开启时受冲击波易破坏。
5. **进 / 排风井紧邻**——污染气体进入新风系统的隐患。
6. **隔墙厚度按非人防墙取**——人防隔墙必须按抗力级别专门设计。
7. **柴油发电机房没作为独立防护区**——油机间运行噪音大、振动大、热源集中，必须独立。
8. **平战转换图纸缺失**——只画战时图或只画平时图，没说明如何转换。
9. **覆土厚度后期被景观改动**——结构计算依据的覆土厚度，绿化景观调整后未通知结构复核。
10. **商业人防中战时口部被店铺占用**——平时招商把战时主要出入口位置改成商铺，紧急情况下出不来。

## 十二、与本 SKILL 内其他清单的关系

- **防火**：人防区在平时使用时仍需符合防火规范（GB 50016）。两套规范的"安全出口数量"、"疏散距离"等可能不同，**取严**。
- **无障碍**：人防防护区战时不要求无障碍；但平时使用部分（如平时商业、车库通道）须满足无障碍。
- **绿建**：人防作为地下空间利用，部分绿建评分项（地下空间利用率、土壤热稳定性等）会涉及，本审查独立进行。
- 与施工质量验收、消防验收并行——人防有专门的人防工程质量监督和竣工验收程序。
