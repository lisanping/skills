# 绿色建筑评价审查清单（GB/T 50378）

> **何时读取此文件**：当用户要求生成或应用绿色建筑评价清单时，包括控制项核查与评分项预评估。

---

## ⚠️ 版本与适用性警告

| 项目 | 说明 |
|------|------|
| 主要标准 | GB/T 50378《绿色建筑评价标准》 |
| 本文件参照版本 | 2019 版（与 2014 版结构差异较大；2019 版引入"以人为本"框架） |
| **使用前必须确认** | 项目实际使用的版本（**新版若已发布，以新版为准**） |
| 配套文件 | 当地绿建评价细则（如《上海市绿色建筑评价标准》）；项目方案阶段确定的**绿建星级目标** |
| 注意事项 | 本标准为**推荐性**（GB/T），但许多省市已纳入强制要求；最终评价由具备资质的第三方评价机构执行 |

**SKILL 在使用前**：

1. 询问项目目标星级（基本级 / 一星 / 二星 / 三星）——直接决定评分项要求
2. 询问建筑类型（公共建筑 / 居住建筑——评分细则不同）
3. 询问设计阶段（方案 / 初步设计 / 施工图——影响可获评分项的范围）
4. 提醒用户：**本清单为自评用途**，不替代第三方评价机构的正式评价

## 一、本清单的组织结构

GB/T 50378-2019 的评价体系：

```
绿色建筑评价
├── 控制项（全部必须满足）
└── 评分项（5 大类 + 加分项）
    ├── 安全耐久（Q1）
    ├── 健康舒适（Q2）
    ├── 生活便利（Q3）
    ├── 资源节约（Q4）
    ├── 环境宜居（Q5）
    └── 提高与创新加分项（QA）
```

### 星级评定逻辑

| 星级 | 控制项 | 总分 | 各类评分项最低得分率 |
|------|-------|------|---------------------|
| 基本级 | 全部满足 | ≥ 60 分 | – |
| 一星级 | 全部满足 | ≥ 60 分 | 各类评分项 ≥ 30% |
| 二星级 | 全部满足 | ≥ 70 分 | 各类评分项 ≥ 30%；同时围护、节能、节水有附加要求 |
| 三星级 | 全部满足 | ≥ 85 分 | 各类评分项 ≥ 30%；附加要求更严 |

> **关键**：星级评定不仅看总分。**任何一类评分项得分率不足 30% 都会拉低整体评级**。审查时不能只算总分。

### 检查项的特殊表格结构

绿建清单与防火 / 无障碍 / 人防的清单**列结构不同**——增加"类别"和"分值"两列：

| 序号 | 条款 | 检查项 | 类别 | 满分 | 限值 / 要求 | 设计值 | 判定 / 得分 | 备注 |
|------|------|--------|------|------|------------|--------|-----------|------|

- **类别**：控制项 / 评分项
- **判定 / 得分**：
  - 控制项：☑ 满足 / ☐ 不满足 / ⚠ 需进一步信息
  - 评分项：实际得分（0 至满分；部分条款支持小数）

## 四、得分汇总与星级判定

### 汇总公式

总得分按下式计算（GB/T 50378-2019 5.2.1）：

```
Q = (Q0 + Q1 + Q2 + Q3 + Q4 + Q5 + QA) / 10
```

其中 Q0 = 控制项基础分（满足时为 400），Q1–Q5 为各评分项实际得分，QA 为加分项得分。

> 注：以上为简化表达，实际公式以规范规定为准。

### 各类评分项得分率（关键！）

```
得分率 = 该类实际得分 / 该类满分 × 100%
```

**任一类得分率不足 30%**，整体最高仅能评为基本级（即使总分超过 70）。

### 判定流程

```
1. 控制项是否全部满足？
   ├── 否 → 不通过任何级别，停止
   └── 是 → 继续 2

2. 总分计算
3. 各类得分率计算
4. 判定星级（按上方"星级评定逻辑"表）
5. 输出最终结果
```

## 六、常见错误（与绿建专项相关）

1. **只算总分不看各类得分率**——总分 75 分但 Q5 环境宜居得分率 25%，依然只能评基本级。
2. **控制项缺失就跳过**——任何一条控制项不满足都停止评价，不能"先看看能拿多少分"。
3. **节能控制项与节能专项审查混淆**——绿建的节能控制项要求"满足节能设计标准"，是节能审查通过即可；评分项才看"优于多少"。两者不要混。
4. **本地化建材统计失真**——500km 半径内生产，要看实际产地证明，不是品牌总部所在地。
5. **装配式得分按"已采用装配式"判而非按"装配率"打分**——装配率有具体计算公式，不能笼统判定。
6. **加分项过度依赖**——加分项总分有上限（一般 100 分），且每条都有具体准入条件，不能简单"叠加"。
7. **方案阶段评价 vs 施工图阶段评价混用**——方案阶段不可获取的评分项（如运行能耗）不能预判得分。
8. **遗忘地方加严**——上海、北京、广州等地有地方版绿建标准，部分条款比国标严。

## 七、与本 SKILL 内其他清单的关系

- **防火**：绿建评分项中部分涉及消防（如疏散安全），但**不替代防火专项审查**——绿建审查中"通过"不等于消防审查通过。
- **无障碍**：绿建有"无障碍步行系统"控制项，但是基本要求；详细审查仍按 GB 50763 进行。
- **人防**：相对独立，但人防工程作为地下空间利用对绿建评分项有贡献（地下空间利用率等）。
- **节能审查**：节能是绿建评分的核心子项之一，但**节能专项审查**（按 GB 50189 / JGJ 134 等）是独立流程，本清单不替代。
