> **何时读取此文件**：当用户要求生成或应用建筑设计防火审查清单时，特别是涉及防火分类、防火分区、安全疏散、构件耐火、消防设施、消防车道等检查项。

## 二、FS-1　建筑分类与耐火等级

### 检查目的

后续大部分条款（防火分区面积、疏散距离、构件耐火极限）都依赖建筑分类。**这是清单的入口**，分类错误会导致整份清单的限值都用错。

### 检查项

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| FS-1.1 | GB 50016 5.1.1 | 民用建筑分类（高层 / 多层 / 单层）核定 | 按建筑高度划分：住宅 27m / 公建 24m 为分界 | | | 高度按规范定义计算（屋面以下） |
| FS-1.2 | GB 50016 5.1.1 | 高层公建分类（一类 / 二类）核定 | 一类高层包括：高度 >50m 的公建、高度 24–50m 但有特定功能（医院、商业、电视塔等） | | | 不能漏判一类——一类要求显著严于二类 |
| FS-1.3 | GB 50016 5.1.3 | 耐火等级核定 | 一类高层公建 ≥ 一级；二类高层公建 ≥ 二级；多层公建按建筑类型 | | | |
| FS-1.4 | GB 50016 5.1.2 | 多功能组合建筑的分类原则 | 同一建筑两种或两种以上使用功能时，按各部分中危险性最大的确定整体类别 | | | 综合体常见，注意商业 + 办公 + 酒店组合 |

### 关键判断：高度计算

GB 50016 中"建筑高度"的算法**不等同于**结构标高或建筑面层标高：

- 平屋面：室外设计地面至屋面面层
- 坡屋面：室外设计地面至檐口或屋脊（按规范取值）
- 不计入高度的部分：局部突出屋面的瞭望塔、冷却塔、设备间、电梯机房、楼梯出口小间（须满足面积比例和高度限制）

**SKILL 在判定 FS-1.1 / 1.2 时**：要求用户明确提供"防火规范意义下的建筑高度"，不要用结构总高或层数乘以层高估算。

## 四、FS-3　平面布置与防火分区

### 检查目的

防火分区是限制火灾蔓延的核心手段，最大允许建筑面积**强制性条款**。

### 防火分区限值（关键查表）

民用建筑每防火分区最大允许建筑面积（一/二级耐火等级），单层 / 多层 / 高层情况不同：

| 建筑类别 | 防火分区最大允许面积（基础） | 设自喷可放大至 |
|---------|---------------------------|---------------|
| 一、二级耐火等级单层、多层公建 | 2,500 m² | 5,000 m² |
| 一类高层公建（标准层） | 1,500 m² | 3,000 m² |
| 二类高层公建（标准层） | 1,500 m² | 3,000 m² |
| 高层裙房（≤24m 部分，与主楼分隔） | 2,500 m² | 5,000 m² |
| 商店、展览、地下商店等 | 见专门条款 | 见专门条款 |
| 地下室、半地下室 | 500 m² | 1,000 m² |
| 地下商店总建筑面积 > 20,000 m² | 须采取下沉式广场 / 防火隔间等措施 | |

> **重要**：上表为通用值。歌舞娱乐、医院病房、托幼、老年人照料等场所有更严格限值；地下设备用房、车库等有专门规范。**SKILL 必须根据具体功能查表**，不能一律套用上表。

### 检查项

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| FS-3.1.1 | GB 50016 5.3.1 | 各防火分区面积是否在限值内 | 见上表，按建筑类型选取 | | | 确认是否设置自喷以适用放宽 |
| FS-3.1.2 | GB 50016 5.3.4 | 商店营业厅、展览厅按设功能合并最大允许面积 | 同 5.3.1 但适用专门条款 | | | |
| FS-3.1.3 | GB 50016 5.4.4 | 歌舞娱乐放映游艺场所平面布置 | 不应布置在地下二层及以下，地下一层地面与室外出入口高差不应大于 10m，且面积有专门限值 | | | 高频违规项 |
| FS-3.1.4 | GB 50016 5.4.5 | 医院、疗养院的病房楼布置 | 病房楼内每个护理单元的建筑面积不应大于专门限值；护理单元应有 ≥ 2 个安全出口 | | | |
| FS-3.1.5 | GB 50016 5.4.4A | 老年人照料设施位置与层数限制 | 独立建造时，建筑高度限制；与其他功能合建时，应布置在独立的防火分区内 | | | 2018 修订强化条款 |
| FS-3.1.6 | GB 50016 5.3.5 | 中庭防火措施 | 中庭与周围相连通的空间应防火分隔（防火卷帘、防火玻璃墙、防火隔间等） | | | 中庭面积应叠加计入总分区面积 |

### FS-4.2　疏散距离

公共建筑房间内最远点至房间门、房间门至最近安全出口的距离限值（一/二级耐火等级、设自喷修正后）：

| 建筑类别 | 房间内最远点至门 | 位于两个安全出口之间 | 位于袋形走道两侧或尽端 |
|---------|----------------|-------------------|-------------------|
| 高层旅馆、医院 | 15m | 30m | 15m |
| 高层办公、其他高层公建 | 15m | 40m | 22m |
| 多层公建（除幼儿园、老年人等） | 22m | 40m | 22m |
| 设自喷的，相应限值 × 1.25 | | | |

> 上表为常用框架，**实际限值随功能、是否设自喷、走道是否安全可控等条件变化**。SKILL 应让用户提供：(1) 是否设自喷，(2) 走道是否敞开式，(3) 房间是否门正对安全出口。

## 十、常见错误（与防火专项相关）

1. **建筑分类错误传递**——一类高层判成二类，导致后面所有限值都用错。FS-1 必须先做且做正确。
2. **是否设自喷未澄清就给数值**——同一条疏散距离，设/不设自喷限值差 25%。
3. **混淆"袋形走道"与"两个安全出口之间"**——疏散距离限值不同。袋形 = 房间门只能向一个方向疏散。
4. **多功能合建未做防火分隔判定**——综合体最常见。要核对各功能分区是否独立、连接处是否有规范要求的防火分隔。
5. **耐火等级与构件耐火极限对应错乱**——结构图上柱梁耐火极限低于规范要求是审图常见漏洞。
6. **登高场地"画在图上但被绿化树遮挡"**——平面图合规，立面/总图与绿化设计冲突。
7. **疏散门反向开启**——尤其是合用前室门、楼梯间门，要求向疏散方向开启。
8. **防火墙未到顶 / 未到屋面板**——防火墙必须从底封到顶并高出屋面（按规范），仅做到吊顶高度无效。
9. **管道井每层未独立封堵**——通用错误，特别是电气竖井。
10. **专项规范覆盖项未识别**——如车库引用 GB 50067 而非 GB 50016 5.3.1，中庭、地下商店、老年人设施都有专门条款。SKILL 应在 FS-1 完成后判断是否需要切换到专项规范。

## 十一、与本 SKILL 内其他清单的关系

- **无障碍**：与防火"疏散"有交叉（如疏散楼梯是否考虑无障碍），但限值各自独立。两份清单分别审查，结果汇总时注意一致性（例如：无障碍坡道不能违反防火间距）。
- **人防**：与防火的关系是"叠加约束"。人防口部、防护单元有自己的疏散和分区要求，必须同时满足防火规范和人防规范——不能用一方限值替代另一方。
- **绿建**：基本独立，但绿建评分项中可能包含"超出强制条款的防火措施"，审查时把它作为加分项而非合规项。
