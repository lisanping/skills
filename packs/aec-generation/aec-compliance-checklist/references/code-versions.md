# 规范版本与修订记录

> **何时读取此文件**：当 SKILL 需要确认或核实规范版本、判断条款是否已修订、提示用户验证版本时。

---

## ⚠️ 此文件本身的有效性警告

本文件记录的是**起草本 SKILL 时的版本信息**——不是实时数据库。

**建筑规范会修订**。一份长期不更新的版本表会比没有版本表更危险，因为它会让用户以为是当前版本。因此 SKILL 在使用此文件时遵循以下流程：

```
1. 从此文件中读取参考版本
2. 用这个版本作为"询问用户"的起点：
   "本 SKILL 参照 GB 50016-2014 (2018 年修订)，这是您项目当前使用的版本吗？
    如果不是，请告知实际版本——若已发布新版，应以新版为准。"
3. 如有联网搜索能力，并行搜索"[规范号] 现行版本"，对比此文件
4. 如用户提供的版本与本文件不同，使用用户提供的版本
```

**本文件应每 6 个月人工核查一次**，由维护者更新。文件末尾的"最后核查日期"是关键元数据。

## 一、规范版本与修订情况

### 1.1 防火相关

| 规范号 | 名称 | 起草本 SKILL 时的现行版本 | 上次主要修订 | 备注 |
|-------|------|------------------------|------------|------|
| GB 50016 | 建筑设计防火规范 | 2014 版（含 2018 年局部修订） | 2018 年（局部修订） | 修订涉及高层建筑、老年人照料设施等条款 |
| GB 50067 | 汽车库、修车库、停车场设计防火规范 | 2014 版 | 2014 年 | 适用于汽车库专项 |
| GB 50229 | 火力发电厂与变电站设计防火规范 | 2019 版 | 2019 年 | 工业项目相关 |
| GB 51251 | 建筑防烟排烟系统技术标准 | 2017 版 | 2017 年 | GB 50016 配套标准 |

**重点关注**：

- GB 50016 在 2018 年的局部修订，强化了老年人照料设施的防火要求（建筑高度限制、防火分隔、避难间设置等），SKILL 在审查老年人照料设施时**必须确认是否已应用 2018 修订内容**。
- 部分省市（如北京、上海）有地方防火设计审核要点，对部分条款做地方加严。

## 二、通用规范与专项规范的关系

近年（2021 年起）住建部发布了一系列**强制性通用规范**（GB 55XXX 系列），这些规范的特点：

- **强制性**——任何建筑都必须满足
- **覆盖面广**——整合多个原专项规范的强制条文
- **优先级高**——与原专项规范冲突时，以通用规范为准

主要通用规范：

| 规范号 | 名称 | 替代 / 整合的旧规范 |
|-------|------|------------------|
| GB 55015-2021 | 建筑节能与可再生能源利用通用规范 | GB 50189、JGJ 134、JGJ 26 等的强条 |
| GB 55037-2022 | 建筑防火通用规范 | GB 50016 等的部分强条 |
| GB 55008-2021 | 混凝土结构通用规范 | GB 50010 等的强条 |
| GB 55021-2021 | 既有建筑鉴定与加固通用规范 | |
| GB 55020-2021 | 建筑给水排水与节水通用规范 | |
| GB 55027-2022 | 城乡排水工程项目规范 | |

**SKILL 使用通用规范的方式**：

```
1. 先看通用规范（GB 55XXX）是否对当前条款有强制要求 → 必须满足
2. 再看专项规范（GB 50XXX）的相应条款 → 通常不冲突，可作为补充判定依据
3. 冲突时以通用规范为准
```

## 三、地方标准

中国大部分一二线城市有地方建筑设计标准，部分内容比国标严或更细。**SKILL 必须询问用户项目所在城市，并提示是否需要叠加地方标准审查**。

常见地方标准：

| 城市 | 例子 |
|------|------|
| 北京 | 《北京市建设工程消防设计审核要点》 |
| 上海 | 《上海市建筑防火设计规则》、《上海市绿色建筑评价标准》DG/TJ 08-2090 |
| 广东 / 深圳 | 《广东省建筑防火设计要点》、《深圳市绿色建筑评价标准》 |
| 江苏 | 《江苏省民用建筑设计防火规范修订意见》 |
| 浙江 | 《浙江省消防技术规范疑难问题操作技术指南》 |

地方标准的获取一般通过当地住建主管部门官网或建筑行业协会。**SKILL 不应假装知道某地最新地方标准的条文**——必须由用户提供或要求用户上传。

## 四、版本核查的最佳实践

### 4.1 SKILL 对话开始时

```
我目前的清单库参照以下版本（信息以本 SKILL 起草时为准）：
- GB 50016 2014 版（含 2018 年修订）
- GB 50763 2012 版
- GB 50038 2005 版
- GB/T 50378 2019 版
- GB 55015 2021 版（节能通用规范，优先级高）

为了让审查准确，能否确认：
1. 您项目使用的是上述版本，还是已发布的新版？
2. 项目所在地（用于核对地方标准）？
3. 如有近期发布的新规范，烦请告知。
```

### 4.2 审查中遇到具体条款时

如对某条具体条款的现行有效性不确定，**SKILL 应该**：

- ✅ 在条款列表中保留该条目，但在"备注"列写"建议核查现行版本"
- ✅ 给出该条款的**精神和数值**作为参考，但不作为最终判定依据
- ❌ **不应该**简单删除条款，那会让用户错过应当审查的项

### 4.3 当用户告知一个新版本时

```
1. 不要假装"我也知道这个新版"——告诉用户："您提到的 [新版] 我无法核对全文，能否提供
   主要变更内容？或者我可以基于 [本 SKILL 参照的旧版] 给出审查框架，
   关键限值的具体数值请您按新版核对。"
2. 对结构性大改（如 GB/T 50378 2014→2019），如果用户的版本与 SKILL 参照版本差异过大，
   建议用户使用最新更新过的 SKILL 版本。
3. 输出的清单中必须明确标注 "依据规范版本：[用户提供的版本]"，不要使用本 SKILL 的参照版本欺骗用户。
```

## 五、规范修订的常见信号

以下情况下 SKILL 应**主动怀疑**所引用的规范可能已修订：

1. **十年以上未修订的国标**（如 GB 50038-2005、GB 50763-2012）——多数会修订
2. **近期政策变化大的领域**——如双碳、装配式、海绵城市相关
3. **强制性通用规范覆盖的领域**（GB 55XXX 系列）——下层规范可能配套修订
4. **重大事故或行业事件后**——往往带来强制条款的局部修订

如有以上信号，SKILL 应在审查开头就提示用户："此规范距今已较久未确认修订，强烈建议您核实当前是否仍是有效版本。"
