# 无障碍设计审查清单（GB 50763）

> **何时读取此文件**：当用户要求生成或应用无障碍设计审查清单时，包括坡道、出入口、卫生间、电梯、停车、标识等检查项。

## ⚠️ 版本与适用性警告

|------|------|

| 本文件参照版本 | 2012 版 |

## 一、本清单的组织结构

清单按 7 个主题分组：

| 编号 | 主题 | 适用对象 |
|------|------|---------|
| AC-1 | 无障碍出入口 | 全部公建、部分居建 |
| AC-2 | 坡道 | 高差 ≥ 400mm 的部位 |
| AC-3 | 无障碍通道与门 | 全部 |
| AC-4 | 无障碍楼梯与电梯 | 公建（多层 / 高层） |
| AC-5 | 无障碍卫生间与卫生器具 | 公建 |
| AC-6 | 无障碍停车位 | 设有机动车停车位的项目 |
| AC-7 | 无障碍标识与盲道 | 公共空间 |

七列表格结构同《fire-safety-checklist.md》。

## 二、AC-1　无障碍出入口

### 检查目的

无障碍出入口是无障碍设计的"门户"。许多项目做了内部无障碍设施却忘记入口处的高差处理，导致整套体系失效。

### 检查项

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| AC-1.1 | GB 50763 3.3.1 | 公共建筑应至少有一处无障碍主要出入口 | 至少 1 个 | | | 主要出入口而非辅助出入口 |
| AC-1.2 | GB 50763 3.3.1 | 居住建筑（含住宅、宿舍）单元入口 | 单元入口（首层入口）应为无障碍出入口 | | | 高层住宅常被忽略 |
| AC-1.3 | GB 50763 3.3.2 | 无障碍出入口形式 | 平坡出入口 / 同时设台阶和轮椅坡道 / 同时设台阶和升降平台（仅在用地受限时） | | | 优先选平坡或坡道；升降平台是不得已 |
| AC-1.4 | GB 50763 3.3.3 | 平坡出入口的坡度 | 室外 ≤ 1:20；门口室外台阶高差 ≤ 15mm 时可不设坡道 | | | 室外平坡 ≤ 1:20 与坡道 1:12 区分 |
| AC-1.5 | GB 50763 3.3.4 | 出入口的轮椅回转空间 | 同时容纳轮椅与陪同人员的空间，平面尺寸 ≥ 1.50m × 1.50m | | | 双扇门或弹簧门处尤其注意 |
| AC-1.6 | GB 50763 3.3.4 | 出入口上方应做雨棚 | 应做雨棚 | | | 雨棚遮挡轮椅回转空间是常见错误 |
| AC-1.7 | GB 50763 3.3.4 | 出入口室内外地面高差 | ≤ 15mm | | | 实际控制在 0–10mm 较安全 |

## 三、AC-2　坡道

### 关键参数（最常考查）

| 项目 | 限值 |
|------|------|
| 轮椅坡道最大坡度 | **1:12**（即每 100cm 升高不超过 8.33cm） |
| 单段坡道最大高差 | **750mm**（坡度 1:12 时，对应水平长度 9.0m） |
| 单段坡道最大水平长度 | **9.0m**（坡度 1:12 时） |
| 不同坡度对应限值 | 1:16 → 高差 ≤ 900mm，水平 ≤ 14.4m；1:20 → 高差 ≤ 1500mm，水平 ≤ 30.0m |
| 坡道净宽 | 室内 ≥ 1.00m；室外 ≥ 1.50m |
| 起点、终点和中间休息平台深度 | ≥ 1.50m |
| 双侧扶手高度 | 上层 0.85m，下层 0.65m（双层扶手） |

### 检查项

| 序号 | 条款 | 检查项 | 限值 / 要求 | 设计值 | 判定 | 备注 |
|------|------|--------|------------|--------|------|------|
| AC-2.1 | GB 50763 3.4.2 | 轮椅坡道最大坡度 | 1:12 | | | 大于 1:12 是绝对不通过 |
| AC-2.2 | GB 50763 3.4.4 | 单段坡道最大高差和水平长度 | 见上表 | | | 超出须设休息平台 |
| AC-2.3 | GB 50763 3.4.5 | 坡道净宽 | 室内 ≥ 1.00m，室外 ≥ 1.50m | | | 净宽不含扶手 |
| AC-2.4 | GB 50763 3.4.5 | 坡道起 / 终点和休息平台深度 | ≥ 1.50m | | | 进出门处需在平台之外另留 1.50m |
| AC-2.5 | GB 50763 3.4.7 | 双侧扶手 | 应设双侧扶手；扶手高度上层 0.85m，下层 0.65m | | | 单侧扶手是常见错误 |
| AC-2.6 | GB 50763 3.4.5 | 临空侧应设安全阻挡（高度 ≥ 50mm） | ≥ 50mm | | | 防止轮椅前轮滑出 |
| AC-2.7 | GB 50763 3.4.6 | 坡道两侧立柱及扶手延伸 | 起点、终点处扶手水平延伸 ≥ 0.30m | | | |
| AC-2.8 | GB 50763 3.4.5 | 坡道地面应防滑 | 防滑材料；变向处应有提示 | | | |

## 十、常见错误（与无障碍专项相关）

1. **居住建筑被遗忘**——很多设计师以为"无障碍只看公建"。实际上住宅单元入口、首层共用部位都有无障碍要求。
2. **入口处理只到门口**——门内地面与门外平坡相连，但门内突然出现 30mm 门槛，是高频错误。
3. **平坡出入口和坡道混淆**——平坡 ≤ 1:20，坡道 ≤ 1:12，标准不同。室外 ≤ 1:20 时不需做"坡道"。
4. **坡道做了但没扶手 / 单侧扶手**——坡道必须双侧扶手，且双层（上 0.85m + 下 0.65m）。
5. **无障碍卫生间面积够但门内开**——门内开会阻挡轮椅；规范要求外开或推拉。
6. **无障碍电梯轿厢内无镜子**——审图常被检出。
7. **无障碍停车位位置不当**——放在车库远端、靠墙角，违反"靠近无障碍出入口"要求。
8. **盲道在路口或出入口断头**——施工和后期维护中尤其常见。
9. **报警装置未接物业系统**——卫生间安装紧急按钮但不连物业台，按了没用。
10. **"少量样本通过"误以为整体通过**——审查必须覆盖所有规范要求的位置（每层、每个公共卫生间），不能抽样。

## 十一、与本 SKILL 内其他清单的关系

- **防火**：疏散楼梯不强制做无障碍楼梯（疏散主要靠消防电梯和楼梯间），但消防电梯的轿厢和候梯厅尺寸必须同时满足无障碍要求。两份清单的电梯条款应交叉核对。
- **人防**：无障碍要求不进入战时人防防护单元，但平时使用部分（地下商业、停车）仍须满足。
- **绿建**：绿建评分项中包含"室外公共空间无障碍设计"等加分项，可作为参考但不替代本审查。
